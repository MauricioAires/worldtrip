### Tecnologias

As seguintes tecnologias foram utilizadas no projeto:

### Requerimentos

- [Node](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/lang/en/)

### Começando

```bash
# Clone this project
$ git clone https://github.com/MauricioAires/worldtrip

# Access
$ cd worldtrip

# Install dependencies
$ yarn install

# Run the project
$ yarn dev
```
